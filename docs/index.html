Your computer:
Part 1 Set it up · you only do this once

Get everything ready

This part takes about 10 minutes. Once it's done, you never have to do it again — downloading later takes seconds.

1

Install Python

Python is the free program that runs the downloader. You install it once.

Go to python.org/downloads and click the big yellow Download Python button. Open the file you downloaded.

⚠️

On the first install screen, tick the box “Add python.exe to PATH” at the bottom before clicking Install. This one checkbox saves a lot of trouble later.

Go to python.org/downloads, click Download Python, open the downloaded .pkg file, and click through the installer.

Most Linux systems already have Python. If not, install it with your package manager — for example Fedora: sudo dnf install python3, or Ubuntu/Debian: sudo apt install python3 python3-venv.

2

Install FFmpeg (strongly recommended)

FFmpeg is a helper that makes proper MP3 music files and unlocks higher-quality video. The downloader still runs without it, but audio comes out as a .m4a file instead of .mp3.

The easiest way is to open the black command window (you'll learn how in Step 4) and paste this one line:

> winget install Gyan.FFmpeg

Prefer not to use a command? You can skip FFmpeg for now and add it later.

Mac installs FFmpeg through Homebrew. If you don't have Homebrew, install it first from brew.sh, then run:

$ brew install ffmpeg

Fedora: enable RPM Fusion first (FFmpeg lives there), then install:

$ sudo dnf install ffmpeg

Ubuntu/Debian: sudo apt install ffmpeg

3

Download the app’s files

Go to the project page: github.com/chukynya/yt-downloader.

  • Click the green <> Code button.
  • Choose Download ZIP.
  • Find the ZIP in your Downloads folder and unzip it (right-click → Extract).

You'll get a folder called yt-downloader. Remember where it is — you'll open it in the next step.

4

Open the command window inside that folder

This is the “black window” where you type simple commands. Opening it from inside the folder matters.

  • Open the yt-downloader folder in File Explorer.
  • Click once in the address bar at the top (where the folder path is shown).
  • Type cmd and press Enter.

A black window opens, already pointing at your folder. 🎉

  • Open the Terminal app (press ⌘ + Space, type “Terminal”, press Enter).
  • Type cd (the letters c-d and a space), then drag the yt-downloader folder onto the Terminal window and press Enter.
  • Open your file manager and go into the yt-downloader folder.
  • Right-click an empty area and choose “Open Terminal Here” (the wording varies by system).
5

Install the app’s parts

In that window, copy the line below, paste it, and press Enter. It sets up a tidy private workspace and downloads what the app needs. Wait until it finishes.

> python -m venv .env && .\.env\Scripts\python.exe -m pip install -r requirements.txt$ python3 -m venv .env && .env/bin/python -m pip install -r requirements.txt
📋

To paste into the black window: right-clickpress ⌘ + Vpress Ctrl + Shift + V. Then press Enter and let it run — it may take a minute.

Part 2 Download something · every time

Grab a video or a song

Setup done! From now on, this is all you do — about 30 seconds.

1

Start the downloader

Open the command window inside the yt-downloader folder again (same as Step 4 above), then paste this and press Enter:

> .\.env\Scripts\python.exe main.py$ .env/bin/python main.py
2

Paste the YouTube link

Copy the web address of the YouTube video from your browser, paste it into the window, and press Enter.

✂️

Pasted a playlist or “radio mix” link with lots of extra text? No problem — the app automatically trims it down and grabs only the one video you're watching.

3

Choose what you want

The app asks a couple of simple questions. Type the number and press Enter — or just press Enter on its own to accept the recommended choice.

🎬

Video (MP4)

The full video with picture and sound. Recommended quality is 1080p (Full HD).

Type 1
🎵

Music (MP3)

Just the audio, as a music file. Recommended quality is 320 kbps (the best MP3 can do).

Type 2
4

Find your file

When it says “completed successfully”, you're done. Your video or song is waiting in a folder named downloads, right inside the yt-downloader folder. Enjoy! 🎉

Stuck?

If something goes wrong

The most common little bumps and how to fix them. Tap a question to open it.

It says “python is not recognized” ›

Windows: Python was installed without the PATH box ticked. Re-run the Python installer, choose “Modify”, and make sure “Add python.exe to PATH” is on — or simply reinstall and tick it (Step 1).

Mac / Linux: use python3 instead of python.

My music saved as .m4a, not .mp3 ›

That means FFmpeg isn't installed yet. Do Step 2, then run the download again and you'll get a proper .mp3.

It says the video is unavailable or shows an error ›

YouTube changes often. Update the app's engine by opening the command window in the folder and pasting:

.\.env\Scripts\python.exe -m pip install -U yt-dlp.env/bin/python -m pip install -U yt-dlp

The window flashes and closes, or nothing happens ›

Make sure you opened the command window inside the yt-downloader folder (Step 4). If you're not sure, close it and do Step 4 again before running the start command.

Is any of this illegal or unsafe? ›

The tool itself is a normal open-source program. Please only download content you have the right to — your own videos, royalty-free material, or where the creator allows it — and respect YouTube's Terms of Service.
